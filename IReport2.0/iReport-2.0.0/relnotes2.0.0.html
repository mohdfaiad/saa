iReport Release 2.0.0
June 18, 2007

The following changes have been made to software since release 1.3.3
  • Full support for JasperReports 1.3.4
  • New XML field mapping tool
  • New document structure outline
  • Improved refactoring capabilities
  • Improved report query dialog
  • Improved UI with sortable tables
  • Several bug fixes
Contributors this time:
  • Giulio Toffoli
Thanks for your suggestions and your on going support using iReport!

Giulio
